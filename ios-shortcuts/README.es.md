# 📱 iOS Shortcuts — Automatizaciones con Scriptable

🌍 [Read in English](README.md) | [Leer en Español](README.es.md)

Los scripts de esta carpeta se ejecutan dentro de la app **[Scriptable](https://scriptable.app/)** en iOS. Pueden dispararse manualmente, integrarse con la app Atajos de Apple, o mostrarse como widgets en la pantalla de inicio.

## Scripts

| Script | Descripción |
|---|---|
| `network_refresh.js` | Verificación de accesibilidad HTTP con soporte para notificaciones y widget |

## Cómo Instalar

1. Instala **Scriptable** desde el [App Store](https://apps.apple.com/app/scriptable/id1405459188).
2. Copia el contenido del archivo `.js`.
3. Abre Scriptable → toca **+** → pega y guarda.
4. *(Opcional)* Agrégalo como widget en la pantalla de inicio: mantén presionado → Editar pantalla de inicio → Agregar widget → Scriptable.
