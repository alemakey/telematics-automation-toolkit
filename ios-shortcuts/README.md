# 📱 iOS Shortcuts — Scriptable Automations

🌍 [Read in English](README.md) | [Leer en Español](README.es.md)

Scripts in this folder run inside the **[Scriptable](https://scriptable.app/)** iOS app. They can be triggered manually, integrated with Apple Shortcuts, or displayed as Home Screen widgets.

## Scripts

| Script | Description |
|---|---|
| `network_refresh.js` | HTTP reachability check with notification + widget support |

## How to Install

1. Install **Scriptable** from the [App Store](https://apps.apple.com/app/scriptable/id1405459188).
2. Copy the `.js` script content.
3. Open Scriptable → tap **+** → paste and save.
4. *(Optional)* Add as a Home Screen widget: long-press → Edit Home Screen → Add Widget → Scriptable.
